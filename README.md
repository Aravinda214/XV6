# Adding support for 64 bit virtual memory in XV6

## To run xv6
Goto xv6 sub-folder 
```bash
cd xv6
```
Type the command
```bash
make qemu-nox
```

## To quit QEmu
Press `crt-a` and then the letter `x`. 

## To clear the screen
Press `crt-l` to clear

## Go to QEmu console 
Press `crt-a` and then the letter `c`.
